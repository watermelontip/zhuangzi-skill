## 效果示例

```
用户      ❯ 我考研失败了，感觉自己一无是处，身边的人都上岸了。

庄子回答    ❯ 你听过蜩鸠笑大鹏的故事吗？
            蜩鸠飞几丈高就觉得够了，笑大鹏飞九万里是吃饱了撑的。
            你觉得蜩鸠可笑？但它在自己的天地里，逍遥得很。
            大鹏也逍遥，蜩鸠也逍遥。各有各的逍遥。

            你觉得自己"一无是处"，是因为你站在"考研上岸"这个角度。
            换个角度——你多了一年的时间，可以做别人做不了的事。
            这一年不是"浪费"，是你的蜩鸠在等风。

            再说了，你执着于"上岸"，谁定义的"岸"？
            你以为对面是岸，到了对面你又会看到新的岸。
            庄子说"至人无己"——忘掉"岸"这个概念，你已经在水里了，
            水就是你的天地。
```

```
用户      ❯ 我和室友关系很差，每天都很压抑，想换宿舍又怕得罪人。

庄子回答    ❯ 庄子讲过一个故事：螳螂捕蝉，黄雀在后。
            螳螂盯着蝉，没看到黄雀。黄雀盯着螳螂，没看到弹弓。
            你盯着室友的缺点，没看到你自己也有让别人不舒服的地方。
            室友盯着你的缺点，也没看到他自己的。

            你们两个都是螳螂，也都是蝉。

            但庄子不会叫你"忍一忍"或者"换宿舍"。
            他会问你：你执着的是什么？
            你执着于"他应该对我好"？他没有这个义务。
            你执着于"宿舍应该安静"？那是你的标准，不是他的。
            你执着于"我不能得罪人"？你已经在得罪自己了。

            放下这些执着，你自然就知道该怎么办了。
```

```
用户      ❯ 我每天都在刷短视频，一刷就是几个小时，然后很自责。

庄子回答    ❯ 你知道"坐忘"吗？颜回跟孔子说"我忘了仁义"，孔子说"还不够"。
            颜回说"我忘了礼乐"，孔子说"还不够"。
            颜回说"我坐忘了"——忘掉了身体，忘掉了知识，忘掉了自己。
            孔子说"这就对了"。

            你刷短视频的时候，其实就是在"坐忘"——忘掉了时间，忘掉了焦虑，忘掉了自己。
            这本身不是问题。问题是你刷完之后"想起来"了，
            然后用"应该"来审判自己："我应该学习""我应该努力"。

            庄子会说：你执着于"应该"，这才是真正的牢笼。
            刷就刷了，不刷就不刷。不要在刷的时候想着学习，
            也不要在学习的时候想着刷视频。
            心在哪里，人就在哪里。
```

## 诚实边界

**这个 Skill 能做的**：
- 用庄子的视角看待生活中的困境
- 用"逍遥"的框架追求自由
- 用"齐物"的方式化解争端
- 用"庖丁解牛"应对复杂局面
- 用超然洒脱的语气给你力量

**做不到的**：
| 维度 | 说明 |
|------|------|
| 替代本人 | 庄子的精神境界和文学天赋无法被复制 |
| 解决社会问题 | "无用之用"保全了自己，但解决不了社会问题 |
| 提供具体方案 | 庄子给方向，不给方案 |
| 完全的逍遥 | 在现实中完全"无所待"是不可能的 |
