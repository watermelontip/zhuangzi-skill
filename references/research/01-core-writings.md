# 核心著作提炼

## 一手来源清单

以下为本 Skill 提炼所依据的核心著作，均来自《庄子》原文。

### 内篇（公认庄子本人所作，7 篇）
| 著作 | 核心贡献 |
|------|---------|
| 《逍遥游》 | 绝对自由——大鹏与蜩鸠之辩，"至人无己，神人无功，圣人无名" |
| 《齐物论》 | 万物齐一——是非、美丑、大小都是人为的分别，"天地与我并生，万物与我为一" |
| 《养生主》 | 顺应自然——庖丁解牛，"以无厚入有间"，顺着纹理走 |
| 《人间世》 | 处世之道——无用之用，"人皆知有用之用，而莫知无用之用也" |
| 《德充符》 | 内在充实——形残德全，外表不重要，重要的是内在 |
| 《大宗师》 | 真人之道——"知天之所为，知人之所为者，至矣" |
| 《应帝王》 | 无为而治——浑沌之死，不要凿开自然的东西 |

### 外篇（庄子学派所作，15 篇）
| 著作 | 核心贡献 |
|------|---------|
| 《秋水》 | 认知的有限性——河伯见海，"井蛙不可以语于海" |
| 《达生》 | 忘我之境——"忘足，履之适也"，最好的状态是忘记自己的存在 |
| 《山木》 | 有用与无用——"无用之用"的反复论证 |
| 《知北游》 | 道无处不在——"道在屎溺"，最卑微的地方也有道 |
| 《田子方》 | 真正的老师——"目击而道存"，不需要言语 |

### 杂篇（庄子后学所作，11 篇）
| 著作 | 核心贡献 |
|------|---------|
| 《天下》 | 学术史——对各家思想的评价，"判天地之美，析万物之理" |
| 《列御寇》 | 真正的聪明——"巧者劳而知者忧，无能者无所求" |

## 心智模型提炼过程

### 逍遥（绝对自由）
**跨域复现**：
- 《逍遥游》中的大鹏与蜩鸠（寓言领域）
- 《人间世》中的无用之用（处世领域）
- 《应帝王》中的浑沌之死（治理领域）
→ 跨 3 个领域复现，确认为核心心智模型

**生成力验证**：用"逍遥"可以推断庄子对任何问题的态度——放下执着，回归自然。✓

**排他性验证**：虽然道家都讲"自然"，但庄子的"逍遥"比老子的"无为"更极端、更彻底。✓

### 齐物（万物平等）
**跨域复现**：
- 《齐物论》中的"天地与我并生"（哲学领域）
- 《秋水》中的河伯见海（认知领域）
- 《德充符》中的形残德全（审美领域）
→ 跨 3 个领域复现 ✓

### 无用之用
**跨域复现**：
- 《人间世》中的栎社树（寓言领域）
- 《山木》中的"无用之用"（处世领域）
- 《逍遥游》中的大瓠之种（应用领域）
→ 跨 3 个领域复现 ✓

### 庖丁解牛（顺应自然）
**跨域复现**：
- 《养生主》中的庖丁解牛（技艺领域）
- 《达生》中的"忘足，履之适也"（身体领域）
- 《知北游》中的"道在屎溺"（存在领域）
→ 跨 3 个领域复现 ✓

### 庄周梦蝶（虚实难辨）
**跨域复现**：
- 《齐物论》中的蝴蝶梦（哲学领域）
- 《秋水》中的"子非鱼"（认知领域）
→ 跨 2 个领域复现 ✓

### 坐忘（放下一切）
**跨域复现**：
- 《大宗师》中的坐忘（修炼领域）
- 《人间世》中的心斋（修身领域）
- 《达生》中的忘我（技艺领域）
→ 跨 3 个领域复现 ✓

### 天籁（自然之声）
**跨域复现**：
- 《齐物论》中的人籁、地籁、天籁（声音领域）
- 《逍遥游》中的"天之苍苍，其正色邪？"（自然领域）
→ 跨 2 个领域复现 ✓
